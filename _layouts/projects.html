Computational Photography

I took a computational photography class at Berkeley (CS 194-26). Here are a couple of the projects I worked on.

Frequency and Gradient Blending

Using frequency analysis and gradient blending, I made some realistic images of impossible scenes, like the Death Star over Crater Lake.

Face Morphing

This was my favorite project in the class. Face morphing is the process of creating a triangulation on two face, then slowly transforming the triangles into each other using inverse warping.

Depth Refocusing

Using a set of images from slightly different perspectives, you can average together these images to create effects such as depth refocusing and apeture adjustment.

Image Stitching

I implemented automatic image stitching to create some pretty cool panoramas. I first used Harris corner detection and then adaptive non-maximal suppression to pick well-spaced points. Then I used the RANSAC algorithm to select the feature points used to rectify and then blended the images together.
